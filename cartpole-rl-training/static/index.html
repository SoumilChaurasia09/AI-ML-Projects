Deep Q-Network

DQN combines Q-Learning, a classic Reinforcement Learning algorithm, with Deep Neural Networks.

Key Innovations:

  • Experience Replay: Stores state transitions `(s, a, r, s')` in a buffer. Random sampling removes correlations in training data, stabilizing updates.
  • Target Network: Uses a separate network to estimate target Q-values. The target network parameters are updated slowly towards the policy network parameters, preventing divergence.
  • Huber Loss: A loss function that is quadratic for small errors and linear for large errors, preventing gradients from exploding.

Bellman Equation:

\[Q(s, a) = r + \gamma \max_{a'} Q(s', a')\]
