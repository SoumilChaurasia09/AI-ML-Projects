How Recognition Works

Unlike standard image classification (which places faces into predetermined categories), this project uses a Siamese Neural Network pipeline:

  1. Detection: The custom FaceDetectorCNN localizes the face bounding box.
  2. Embedding Extraction: The cropped face is passed through the FaceEmbeddingCNN, extracting a 128-dimensional vector representation.
  3. Vector Database Search: The vector is compared against previously registered representations using Euclidean Distance. If the vector lies within a threshold radius (e.g. < 0.65), it registers a match.

This vector-embedding approach means you can register new faces dynamically without retraining the entire network!
